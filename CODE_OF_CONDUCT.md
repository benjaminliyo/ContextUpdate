# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and focus discussion on the
work. Harassment, personal attacks, and discriminatory behavior are not
tolerated.

Report concerns to the maintainers via the repository's issue tracker
(privately, by emailing the address listed in `package.json` if the
matter is sensitive).
